# 高职院校教育产品销售全流程辅助

## 概述

面向高职院校教育产品销售人员的专业辅助技能，基于信号驱动筛选、SPIN深度探索、MEDDPICC交易评估、赢主题方案设计、多线关系经营等专业方法论，提供从目标筛选到客户持续经营的10步循环销售流程指导。

**适用业务**：纸质教材出版、数字资源制作（视频/文档/二维三维动画/H5）、数字教材出版。

## 完整流程（10步循环）

```
步骤1：信号驱动的目标筛选与优先级排序
步骤2：深度背景调研与利益相关者预映射
步骤3：制定差异化触达策略与拜访准备
步骤4：多渠道约访与首次接触
步骤5：结构化现场拜访（SPIN需求探索 + 价值传递）
步骤6：交易评估与竞争策略制定（MEDDPICC）
步骤7：多角色关系经营与决策链覆盖
步骤8：方案设计与赢主题构建
步骤9：招投标响应与商务谈判
步骤10：交付跟进与客户持续经营（QBR）
    ↻ 循环回步骤1
```

## 核心方法论速查

### 信号分类（Outbound Strategist）

- **Tier 1 主动购买信号**：招标公告、供应商征集 → 立即行动
- **Tier 2 组织变动信号**：新领导、新荣誉、新项目 → 快速跟进
- **Tier 3 技术行为信号**：教改探索、数字化转型 → 持续培育

### SPIN探索（Discovery Coach）

- **Problem**：让客户说出不满（"老师们对现有资源满意度如何？"）
- **Implication**：揭示不解决的后果（核心发力点，激活损失厌恶）
- **Need-payoff**：让客户描绘解决方案的价值

### MEDDPICC评估（Deal Strategist）

8项逐项评估：Metrics / Economic Buyer / Decision Criteria / Decision Process / Paper Process / Identify Pain / Champion / Competition

- 6-8分确认 → A级重点投入
- 3-5分确认 → B级持续培育
- <3分确认 → C级降低优先级

### 赢主题设计（Proposal Strategist）

公式：**[客户具体挑战] + [我们独特能力] + [可量化结果]**

### 客户经营（Account Strategist）

- 至少维护3条独立关系线
- 每季度QBR回顾
- 从"一次性交易"到"持续合作"

## 高职教育行业关键知识

### 采购周期

- 3-4月：教材选用评审启动（秋季教材推介窗口）
- 5-6月：教材选用结果公示、采购招标
- 10-11月：春季教材选用启动（春季教材推介窗口）
- 11-12月：春季教材采购、年度预算编制

### 决策链条

- **教材选用**：教师/教研室 → 二级学院 → 教务处 → 教材选用委员会 → 院长办公会 → 采购
- **数字资源/信息化**：二级学院/教务处 → 信息中心 → 信息化领导小组 → 采购

### 关键角色

| 角色 | 职能 | 优先级 |
|------|------|--------|
| 教材科/教学建设科 | 日常对接，信息入口 | 最高 |
| 二级学院院长 | 需求方，推荐权（Champion候选） | 高 |
| 教务处长 | 预算分配，决策影响 | 高 |
| 分管教学副校长 | 最终审批（Economic Buyer） | 高（通过引荐） |
| 信息中心主任 | 数字资源项目归口管理 | 中高 |
| 采购部门 | 招标流程把关 | 中 |
