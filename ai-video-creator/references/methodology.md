### 一、视频创作全流程

（从最初的创意构思到最终发布，金属文老师的创作步骤解析）

1. **创意思考与动机**
    - 受优秀作品启发，结合个人兴趣领域确定主题
    - 以赛代练，通过实际项目提升 AI 工具使用技能
    - 结合最新 AI 模型功能，探索创作可能性
2. **音乐制作**
    - 使用 Suno、Udio 等 AI 音乐工具生成背景音乐
    - 根据视频节奏设计音乐结构，注重起承转合
    - 利用分轨功能，便于后期剪辑时灵活调整
3. **脚本与分镜设计**
    - 根据音乐节奏和情绪设计故事框架
    - 边生成素材边完善分镜，保持创作灵活性
    - 注重镜头多样性，避免单一视角导致单调
4. **图像与视频生成**
    - 使用 Midjourney 生成角色设定图，确保正面、背面、全身和特写
    - 优先选择图生视频方式，保证角色和风格一致性
    - 利用首尾帧功能解决复杂动作和视角转换问题
5. **配音与音效**
    - 使用海螺（MiniMax）等工具进行 AI 配音，支持声音克隆
    - 调整语速、音高参数，匹配角色形象和情绪
    - 结合音效库和 AI 生成音效，增强视频沉浸感
6. **剪辑与发布**
    - 根据音乐节奏剪辑视频，卡点增强表现力
    - 统一色调，使用转场效果增强流畅度
    - 优化封面和标题，分析数据持续改进

### 二、推荐 AI 创作工具

- **即梦**：打工皇帝
- **海螺（MiniMax）**：动作指导。AI 配音工具，支持声音克隆、多语种和情绪调节
- **Nano Banana Pro**：改图专家，角色一致性几乎可达到100%。谷歌发布的图像编辑模型，用于优化生成的图像，提升质量
- **Suno / Udio**：AI 音乐生成工具，可生成带歌词或纯音乐，支持分轨，便于后期调整
- **Midjourney**：AI 图像生成工具，用于创建角色设定和场景氛围，支持风格调整
- **通义万象 2.2**：开源视频生成模型，支持文生视频和图生视频，均衡角色稳定性
- **剪映 / 达芬奇**：视频剪辑工具，用于素材拼接、调色、加特效和字幕

### 三、创作技巧与经验

1. **提示词编写技巧**
    - 使用 RTF 结构（角色 - 任务 - 格式）结构化表达
    - 信息尽量全面清晰，减少大模型联想空间
    - 正面表述优于负面表述，明确告知做什么
    - 使用 Markdown 等结构化语法，添加提示强化
    - 风格确定后优先“手动微调逐镜头”，而不是依赖网上提示词/批量扩写
        - 先用少量探索图（3-10张）锁定：角色造型、镜头语言、光线、色调、材质、画幅
        - 固定一条“风格锚点”（style anchor）：只描述风格与摄影语言，不描述剧情内容
        - 每个分镜只改“内容变量”（角色动作/表情/构图/场景要素），风格锚点尽量不动
        - 记录每次修改与结果（prompt log），方便回滚与复用
2. **保持风格一致性**
    - 生成角色正 / 背 / 全身 / 特写四视图作为参考
    - 固定风格提示词，确保场景画风统一
    - 优先使用图生视频，而非文生视频
    - 同一项目尽量使用同一工具生成素材
3. **视频制作技巧**
    - 利用首尾帧功能解决复杂动作和视角转换
    - 镜头连贯：用“上一段视频尾帧”作为“下一段视频首帧”（图生视频/首尾帧都适用）
        - 从上一段导出/截图尾帧（尽量无运动模糊、主体清晰）
        - 下一段生成时：把尾帧作为 init/first frame，并保持相同角色参考图与风格锚点
        - 需要更强控制时：下一段同时设置 last frame（下一镜头的目标画面），中间交给模型补间
    - 多个镜头拼接时注意色调统一，可后期调整
    - 灵活运用不同景别和机位，避免单调
    - 视频生成后进行高清放大处理，提升质量
4. **音乐与剪辑技巧**
    - 先制作音乐再创作画面，根据音乐节奏剪辑
    - 利用音乐分轨，根据画面需要灵活调整乐器
    - 重拍和节奏点处配合画面切换，增强表现力
    - 善用闪黑、闪白和慢动作，保持流畅度

### 四、常见问题解答

1. **如何解决首尾帧视频色调不一致、衔接不顺畅的问题？**
    - 使用 Sora 的 remix 功能，对两段视频的衔接部分重新生成
    - 将前一段视频的结尾作为后一段视频的开篇，保持色调一致
    - 同一项目尽量使用同一工具生成所有视频片段
    - 后期调整过渡速度，使转换更自然
    - 后期色调统一处理，确保色调一致
2. **如何制作有良好起承转合的剧本？**
    - 多看优秀作品，积累不同导演和风格的视觉经验
    - 不必一开始就追求完美脚本，可以边创作边完善
    - 可以参考专业编剧的方法和框架
    - 注重情绪的递进和转折，保持叙事节奏
3. **使用 AI 写提示词有什么技巧？**
    - 构建鲁棒性的方法论，不受模型更新影响
    - 使用 RTF 结构（角色 - 任务 - 格式）进行结构化表达
    - 提供全面清晰的信息，减少 AI 的联想空间
    - 正面表述优于负面表述，明确告知做什么
    - 使用 Markdown 等结构化语法，添加提示强化
    - 遵循奥卡姆剃刀原则，用最少的词表达最清楚意思
4. **剪映是否可以自动识别音乐节奏？**
    - 剪映具有 "一键成片" 等功能，可以实现一定程度的音乐节奏识别和自动剪辑。不过对于专业创作，建议手动根据音乐的重拍和节奏点进行标记和剪辑，这样可以更精准地匹配画面与音乐，营造营造更好的视听效果，尤其是中国传统音乐，重拍位置有其特点，手动处理能获得更佳效果

### 五、关于 AI 视频的思考

AI 视频的终局，其实是内容的终局，从广告到小说、戏曲、电影、游戏视频，但能流传下来的都是足够好的作品。

无论 AI 如何发展，好故事、好音乐、好内容的核心价值不会改变。形式一直在变，但那些能打动人心的好作品都有一个共同点 —— 更好。

找到自己真正热爱的方向，不断进步，让作品越来越好，这才是最坚固的护城河。
