# 热点追踪 (Hot Topic Archetype)

> 来源：7位头部AI博主分析报告综合提炼（卡兹克、赛博禅心、卡尔、黄叔、饼干哥哥、藏师傅、洛小山）

---

## 适用场景

当文章主题是以下任一情况时使用本原型：

- AI 领域重大新闻/发布（新模型、新产品、重大更新）
- 行业事件快速反应（收购、融资、争议、政策）
- 技术突破的第一时间解读
- 圈内热议话题的个人立场表达
- 突发事件的即时评论

**核心原则**：不是新闻通讯社，而是「一个有判断力的人在第一时间告诉你这件事意味着什么」。

---

## 结构模板（含比例分配）

```
事实/事件（10%）
  └─ 发生了什么？用最简洁的方式交代清楚
  └─ 时间、主体、事件三要素

为什么重要（Personal Take）（20%）
  └─ 这件事为什么值得写？
  └─ 不是客观重要性，是「我为什么觉得重要」

深度分析（40%）
  └─ 技术/商业/文化层面的拆解
  └─ 与已有认知的对比（"之前我们以为...现在..."）
  └─ 对行业格局的影响判断

对读者的影响（15%）
  └─ 这件事跟普通读者有什么关系？
  └─ 你现在应该做什么/关注什么？

立场判断（15%）
  └─ 我的最终态度
  └─ 对未来的预判
  └─ 情绪收束（从震惊到理性，从理性到感性升华）
```

## 开头模式（含中文示例）

### 模式一：震惊反应式

用最原始的情绪反应开场，拉近与读者的距离。

> 卧槽？？？
> -- 赛博禅心

**要点**：情绪必须真实，不能为了震惊而震惊。这种开头只适用于真正重大的事件。如果每篇都"卧槽"，就没人信了。

### 模式二：深夜/即时发现

交代发现这件事的具体时间和场景，制造"你跟我同步获取最新消息"的参与感。

> 现在是2点22分...然后，特么的，看到了一个消息。
> -- 卡兹克

**要点**：具体时间 + 具体场景 + 情绪反应三件套。凌晨写作本身就是态度信号——"这件事重要到我睡不着"。

### 模式三：来不及了快上车

制造紧迫感，让读者觉得现在不看就来不及了。

> 来不及解释了，凌晨一点刷到影视飓风的更新。
> -- 卡尔

**要点**：「来不及解释了」本身就是一种速度信号。配合具体时间更有效。

## 逐节写作指导

### 第一节：事实/事件（约10%篇幅）

**目标**：用最少的字数让读者知道发生了什么。

写法要求：
- 一句话交代事件核心（谁 + 做了什么 + 什么时候）
- 如果是产品发布，给出产品名+公司名
- 如果是争议事件，客观陈述双方立场
- 不在这里展开分析，纯事实

**禁忌**：不要用"据报道""根据消息"等第三人称新闻体。用"我刚看到""刚发生了一件事"。

### 第二节：为什么重要（约20%篇幅）

**目标**：建立「这件事值得你花5分钟读」的理由。

写法要求：
- 用第一人称表达：「我觉得这件事重要是因为...」
- 放到历史坐标中：「上次让我有这种感觉的，还是XX的时候」
- 给出判断的理由，而不只是断言
- 可以用类比帮助理解量级（"这相当于..."）

**核心区分**：新闻的"为什么重要"是客观的；热点追踪文章的"为什么重要"是主观的——是你个人的判断和直觉。

### 第三节：深度分析（约40%篇幅 -- 核心部分）

**目标**：提供读者自己想不到的分析角度。

写法要求：
- 至少从两个维度分析（技术/商业/文化/用户体验/竞争格局中选）
- 每个维度内部有递进逻辑
- 用"之前...现在..."的对比框架
- 引入读者可能忽略的信息
- 可以大胆预判，但要标注"这是我的猜测"

**节奏控制**：
- 先给出表层分析（大多数人能想到的）
- 再给出深层分析（需要背景知识才能看到的）
- 最后给出个人洞察（只有你能给出的判断）

### 第四节：对读者的影响（约15%篇幅）

**目标**：回答"so what? 跟我有什么关系？"

写法要求：
- 分人群说明影响（开发者/普通用户/创业者/...）
- 给出具体的行动建议（"现在可以去试试XX""先观望，等XX再说"）
- 如果影响还不确定，诚实说"目前还不好说，但值得关注的是..."

**禁忌**：不要说"让我们拭目以待"这种空话。要么给行动建议，要么给关注方向。

### 第五节：立场判断（约15%篇幅）

**目标**：给出你的最终态度，收束全文情绪。

写法要求：
- 一句话总结你的立场
- 对未来做一个有时间节点的预判（"我觉得半年内..."）
- 情绪收束：从文章开头的震惊/兴奋，回到理性思考
- 可选：上升到更大的人文关怀（卡兹克的「悲悯式」结尾）

**关键技巧**（来自分析报告）：卡兹克最好的热点文章，结尾都会从具体事件跳出来，连接到更大的人类关怀。不是每篇都需要，但重大事件值得这样做。

---

## 必备元素清单

| 元素 | 说明 | 必要性 |
|------|------|--------|
| 第一人称反应 | 不是新闻聚合，是"我"的反应 | 必须 |
| 个人判断/立场 | 对事件的明确态度 | 必须 |
| "Why this matters to YOU" | 对读者的具体影响 | 必须 |
| 速度 vs 深度平衡 | 快不代表浅，深不代表慢 | 必须 |
| 具体产品/公司名 | 不要用"某大厂""某模型" | 必须 |
| 时间锚点 | 写作时间或事件发生时间 | 推荐 |
| 历史对比 | "上次这种级别的事是..." | 推荐 |
| 未来预判 | 带时间节点的预测 | 推荐 |

## 反面模式（Anti-patterns）

### 1. 纯新闻搬运

**错误示范**：
> 据报道，OpenAI于北京时间XX日发布了新模型XX。该模型在XX基准测试中取得了XX的成绩...

**问题**：读者为什么不直接看新闻？你的价值在于判断和解读，不在于信息本身。

### 2. 第三人称新闻体

**错误示范**：
> 业内人士表示，此次发布将对行业产生深远影响...

**问题**：失去了个人写作的核心优势——真实的个人视角。

### 3. 没有判断的结尾

**错误示范**：
> 让我们拭目以待，看看这款产品未来的表现如何。

**问题**：读者读完整篇文章，最需要的是你的判断。"拭目以待"等于没说。

### 4. 照搬新闻稿用语

**错误示范**：
> 此次发布标志着XX领域进入了新的发展阶段，具有里程碑式的意义...

**问题**：公关稿式措辞会让读者瞬间失去兴趣和信任。

### 5. 震惊情绪过度消费

**错误示范**：
（每篇文章都用"卧槽""震惊""颠覆"开头）

**问题**：情绪通胀。真正值得震惊的事才用震惊开头，否则读者会免疫。

## 核心写作技巧

### "加判断"法则

**永远不要只报道发生了什么，永远要加上它意味着什么。**

这是热点追踪原型区别于新闻报道的核心法则。每一个事实陈述后面，都应该跟着你的一句判断。

格式参考：
```
事实 → 所以我觉得... / 这意味着... / 换句话说...
```

### 卡兹克的「悲悯式」结尾

分析报告显示，卡兹克最好的热点文章都有一个共同特征：结尾从具体事件跳出来，连接到更大的人类关怀。

**适用场景**：重大事件、涉及人的命运的话题、行业变革。
**不适用场景**：纯产品发布、功能更新、技术细节。

### 速度与深度的平衡

- **2小时内发出**：重事实+第一反应+初步判断，深度可以后续补充
- **6小时内发出**：需要有独特分析角度，不能只是重复已有观点
- **24小时后发出**：必须有深度洞察，否则热度已过，没有发的必要

## 评分示例段落

### 示例一：高分开头（10/10）

来源：卡兹克

> 现在是2点22分，跟朋友们打完LOL手游，准备倒头就睡。临睡前，刷了一眼X。然后，特么的，看到了一个消息，能给我震惊成傻逼的消息：OpenAI，发他们的文生视频大模型，Sora了。。。。。

**评分理由**：
- 「2点22分」—— 精确到分钟的时间锚点，制造强烈现场感
- 「打完LOL手游，准备倒头就睡」—— 生活场景，不是在办公室等新闻
- 「特么的」—— 真实情绪，不装专业
- 「能给我震惊成傻逼的消息」—— 预告信息量级，吊起读者胃口
- 整段节奏：日常→偶然→震惊，完美的情绪曲线

### 示例二：高分判决（9/10）

来源：赛博禅心

> 此刻，中国团队在通用智能体赛道上，迈出了实质性的第一步。2025，东方破晓。

**评分理由**：
- 「此刻」—— 时间感，把读者拉到历史瞬间
- 「实质性的第一步」—— 精确判断，不是"重大突破"这种空话
- 「2025，东方破晓」—— 四个字的诗意收束，有记忆点
- 从技术事件上升到文明叙事，格局打开但不空洞
- 扣分点：如果前文分析不够扎实，这种结尾会显得浮夸

### 示例三：反面教材（3/10）

> 近日，某知名AI公司发布了其最新的大语言模型。该模型在多项基准测试中表现优异，引发业内广泛关注。业内人士表示，这一发布将对整个行业产生深远影响。让我们拭目以待。

**扣分理由**：
- 「某知名AI公司」—— 连名字都不敢写？
- 「表现优异」—— 优异到什么程度？具体数字呢？
- 「业内人士表示」—— 谁？你自己吗？
- 「深远影响」—— 什么影响？多深？多远？
- 「让我们拭目以待」—— 读完等于没读
- 全篇没有一句个人判断

## 写作节奏参考

| 段落位置 | 情绪节奏 | 信息密度 |
|----------|----------|----------|
| 开头 | 震惊/紧迫/兴奋 | 低（情绪先行） |
| 事实交代 | 冷静/客观 | 高（快速补齐信息） |
| 为什么重要 | 认真/笃定 | 中（建立判断框架） |
| 深度分析 | 理性/专业 | 高（核心价值输出） |
| 对读者影响 | 实用/贴近 | 中（可操作建议） |
| 立场判断 | 坚定/或感性升华 | 低（一两句收束） |
