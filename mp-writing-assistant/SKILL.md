# 公众号写作助手

## 概述

本技能提供公众号文章写作的完整方法论，基于写作五步法，专门针对公众号平台的特点进行优化，包括排版规范、金句提炼、CTA设计等公众号特有的技巧。

## 核心工作流程：公众号写作五步法

```
选题 → 大纲 → 正文 → 标题 → 排版
```

## 步骤1：选题 - 找到值得写的内容

### 核心原则

1. **价值导向**: 解决读者真实问题或提供新视角
2. **受众明确**: 清楚目标读者是谁，他们关心什么
3. **角度独特**: 避免泛泛而谈，找到差异化切入点
4. **可执行性**: 选题要有足够素材支撑，能写完整
5. **时效性**: 结合热点但不盲从，保持独立思考

### 公众号选题特点

- **话题性**: 选择有讨论价值的话题
- **传播性**: 考虑文章的传播潜力
- **垂直性**: 符合公众号定位和受众需求

### 选题检查清单

- [ ] 这个主题能解决什么问题？
- [ ] 目标读者会感兴趣吗？
- [ ] 我的观点或角度有什么不同？
- [ ] 我有足够的素材和案例吗？
- [ ] 这个主题值得花时间深入吗？

## 步骤2：大纲 - 构建清晰的文章骨架

### 核心原则

1. **逻辑递进**: 从问题到方案，从现象到本质
2. **层次分明**: 主论点-分论点-论据，三级结构清晰
3. **重点突出**: 核心观点占60%篇幅，次要内容占40%
4. **开头抓人**: 第一段要能抓住注意力，提出问题或场景
5. **结尾升华**: 总结要点，给出行动建议或启发思考

### 公众号大纲模板

```
H1: 主标题（核心观点）
  H2: 引言（问题/场景）
  H2: 第一部分（核心论点1）
    H3: 分论点1.1
    H3: 分论点1.2
  H2: 第二部分（核心论点2）
    H3: 分论点2.1
    H3: 分论点2.2
  H2: 总结（行动建议/思考）
```

### 大纲检查清单

- [ ] 每个H2是否围绕主论点展开？
- [ ] 逻辑顺序是否合理？
- [ ] 重点内容是否突出？
- [ ] 是否有冗余或缺失的部分？

## 步骤3：正文 - 用文字传递价值

### 核心原则

1. **开门见山**: 第一段直接点题，不要绕弯子
2. **案例支撑**: 每个观点至少配1个具体案例或数据
3. **语言简洁**: 删除冗余词汇，一句话说清楚一件事
4. **节奏把控**: 长短句结合，段落3-5行为宜
5. **情感共鸣**: 用故事和场景让读者产生代入感

### 正文写作技巧

- 用"你"而不是"我们"，增强对话感
- 用具体数字代替模糊表述（"80%"而非"大多数"）
- 用对比突出观点（之前vs之后，错误vs正确）
- 用问句引导思考，增加互动感
- 用金句总结段落，便于记忆和传播

### 金句提炼

**金句特点**：
- 简洁有力，一句话说清楚核心观点
- 易于记忆和传播
- 能够引发共鸣

**金句位置**：
- 段落结尾
- 章节总结
- 文章结尾

### 正文检查清单

- [ ] 每个段落是否围绕一个中心？
- [ ] 案例是否贴切且有说服力？
- [ ] 语言是否简洁易懂？
- [ ] 是否有金句可以提炼？

## 步骤4：标题 - 抓住第一眼注意力

### 核心原则

1. **价值明确**: 一眼看出能获得什么（知识/方法/启发）
2. **数字具体**: 用数字增强可信度（"3个方法"而非"几个方法"）
3. **情绪共鸣**: 击中读者痛点或爽点
4. **悬念适度**: 引发好奇但不故弄玄虚
5. **长度适中**: 公众号标题20-30字最佳

### 标题公式

- **数字型**: "X个方法/技巧/原则..."
- **对比型**: "从X到Y，我学会了..."
- **问题型**: "为什么X总是Y？"
- **场景型**: "当你X时，记住Y"
- **结果型**: "做了X，我得到了Y"

### 标题检查清单

- [ ] 读者能一眼看出价值吗？
- [ ] 是否足够具体和可信？
- [ ] 能引发点击欲望吗？
- [ ] 是否与正文内容匹配？

## 步骤5：排版 - 公众号适配

### 核心原则

1. **三级标题**: 只用 H1/H2/H3
2. **段落分割**: 用'---', 上下必须空行
3. **有序列表**: 数字后无空格 ('1.项目')
4. **行高 1.8**: 书籍标准
5. **克制配色**: 灰色管结构，蓝色管交互，黑色管内容

### 公众号排版规范

- **段落间距**: 段落之间用空行分隔，重要段落用'---'分隔
- **加粗强调**: 关键信息用加粗突出
- **Obsidian 加粗**: `**` 紧贴文字；句号/逗号/冒号等放在 `**` 外，括号/引号视为内容放在 `**` 内
- **引用标注**: 重要观点用引用格式
- **列表格式**: 有序列表和无序列表规范使用
- **插图位点**: 在关键位置预留插图位点
- **图片引用格式**: `![标题](url)` 中方括号内不带扩展名（如 `![技术模型](url.png)` 而非 `![技术模型.png](url.png)`）

### 自动排版处理项（必做）

- **图片标题去扩展名**: 将 `![xxx.png](url.png)` 转为 `![xxx](url.png)`，仅处理方括号内文本

### 排版检查清单

- [ ] 标题层级是否清晰？
- [ ] 段落是否用'---'分隔？
- [ ] 列表格式是否正确？
- [ ] 图片引用方括号内无扩展名（`![标题]` 而非 `![标题.png]`）
- [ ] 整体视觉是否舒适？

## 公众号特有技巧

### CTA（行动号召）设计

**位置**：
- 文章结尾
- 关键转折点
- 章节结尾

**类型**：
- 关注公众号
- 转发分享
- 留言互动
- 点击链接

### 小标题优化

**小标题作用**：
- 引导阅读节奏
- 突出核心观点
- 增强可读性

**小标题技巧**：
- 简洁有力
- 与正文呼应
- 形成逻辑链条

### 插图建议

**插图位点**：
- 文章开头（吸引注意）
- 章节转折（缓解疲劳）
- 关键观点（增强理解）
- 文章结尾（强化记忆）

## 反模式（NEVER Do）

### ❌ 反模式1：选题过于宽泛

**错误做法**: "如何提高工作效率"（太宽泛）
**正确做法**: "3个时间管理技巧，让我的工作效率提升80%"

### ❌ 反模式2：大纲逻辑混乱

**错误做法**: 想到哪写到哪，没有清晰结构
**正确做法**: 先构建三级结构（主论点-分论点-论据）

### ❌ 反模式3：正文缺乏案例

**错误做法**: 只有观点，没有具体案例支撑
**正确做法**: 每个观点至少配1个具体案例或数据

### ❌ 反模式4：标题故弄玄虚

**错误做法**: "震惊！这个方法竟然..."（过度营销）
**正确做法**: "3个方法，解决你的XX问题"（价值明确）

### ❌ 反模式5：排版不规范

**错误做法**: 标题层级混乱，段落不分隔
**正确做法**: 遵循三级标题、段落分隔、列表格式规范

### ❌ 反模式6：忽略CTA设计

**错误做法**: 文章结尾没有行动号召
**正确做法**: 设计明确的CTA，引导读者行动

## 使用示例

**用户**："帮我写一篇关于时间管理的公众号文章"

**响应流程**：
1. **选题**: 确认角度（如"3个时间管理技巧"）
2. **大纲**: 构建三级结构（引言→技巧1→技巧2→技巧3→总结+CTA）
3. **正文**: 每个技巧配具体案例和数据，提炼金句
4. **标题**: 优化为"3个时间管理技巧，让我的工作效率提升80%"
5. **排版**: 检查标题层级、段落分隔、列表格式、插图位点
6. **增强**: 提供2-3句金句，给出排版建议
